# About pyviewfactor

## What is pyviewfactor?

[`pyviewfactor`](https://gitlab.com/arep-dev/pyViewFactor) is a Python library dedicated to the computation of radiative view factors between planar facets.

## Why use it in this session?

It provides a practical framework for:

- computing view factors
- studying visibility and obstruction effects
- connecting geometry with thermal analysis workflows

## Typical use cases

- building simulation
- thermal comfort analysis
- radiative exchange studies
- educational demonstrations
